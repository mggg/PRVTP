# PRVTP
 
Code for PRVTP paper. The code is sorted by section.
- "BT fpv to coh" contains code that converts an expected share of FPV to the correct slate BT cohesion parameter.
- "Disprop" contains the pipeline for making waterfall plots.
- "Generator Marginals" explores the properties of our ballot generators, like what share of ballots have an A candidate ranked second.
- "MDS" generates the MDS plots
- "Scottish" refers to code used to understand real scottish elections, in particular, generating the tables of disprop scores and the attempt at fitting to real world data. 
- "Section 4" refers to the figures used in the section about theorems proved about slate PL.
